# Content Production SOP (Publishing)

## Step 1 — Topic selection
- Pull next topic from keyword_map.md
- Confirm intent (informational vs transactional)

## Step 2 — Draft using templates/blog-template.md
- Must include Quick Answer
- Must include India context
- Must include FAQs

## Step 3 — Internal linking
- Add 1 pillar link + 2 related blog links
- Add to pillar page "Recommended reads"

## Step 4 — Metadata + Schema
- Add title + meta description
- Add BlogPosting + FAQ schema JSON-LD

## Step 5 — QA Checklist
- One H1 only
- No fluff intros
- Lists are accurate + consistent
- Disclaimer included
- Slug is clean and stable

## Step 6 — Publish
- Update sitemap
- Verify in Search Console after deploy
