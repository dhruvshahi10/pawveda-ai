# SEO Information Architecture (IA)

## URL Structure
- Pillars:
  /guides/food-safety
  /guides/climate-safety
  /guides/daily-care
  /guides/breed-intelligence

- Blogs:
  /blog/dogs/<slug>
  /blog/cats/<slug>

## Canonical Rules
- One topic = one page.
- Do NOT mix dogs and cats on the same blog unless the query intent is mixed.

## Page Types
1) Landing Page (conversion)
2) Pillar Page (hub + internal linking)
3) Blog Post (query-focused)
4) Glossary (optional later)
