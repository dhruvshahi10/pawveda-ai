# Page Template Standard

## Required On-Page Elements
- Title tag: 50–60 chars
- Meta description: 140–160 chars
- H1: exactly 1
- H2/H3: structured and skimmable
- FAQ section (3–5)
- Internal links:
  - 1 link up to pillar
  - 2 links sideways to related blogs
- Disclaimer (trust_and_disclaimer_blocks.md)

## Schema Guidelines
- BlogPosting for blogs
- FAQPage for FAQs
- BreadcrumbList recommended

## Media Rules
- 1 hero image minimum
- Alt text must be descriptive (not keyword stuffing)
